# 101zh.github.io

A Profile Website where I store my networking and cybersecurity related labs.\
There may be more for the website in the future, but for now that's it...

## What are all the Write-Ups About?

To sum it up, the write-ups (or reports) are about configuring specific devices to communicate, block, or filter internet traffic. In general they are detail how computers, routers, firewalls, switches communicate.

## Features

- Responsive design, so the website is viewable in both portrait and landscape on *most* browsers
- Animations on important objects in the website
